# Guddoers Media Website

Welcome to the Guddoers Media website project! This is a simple, straightforward static website built with HTML, SCSS (Sass), and Vanilla JavaScript. 

This README provides all the essential information you need to understand the structure of the project and how to update it.

## Project Structure

Here's a quick overview of what the files and folders do:

- **`.html` files (`index.html`, `about.html`, `shop.html`)**: These are the main web pages. Any changes to the content, text, layout structure, or links should be done in these files.
- **`styles/`**:
  - `style.scss`: **This is where you write your styles.** It uses SCSS, which allows for variables, nesting, and mixins.
  - `style.css`: This is the compiled stylesheet that the HTML files actually use. **Do not edit this file directly**, as your changes will be overwritten the next time the SCSS compiles.
- **`scripts/script.js`**: Contains the JavaScript code for the website, such as handling the mobile navigation menu and the contact form submission.
- **`assets/`**: Contains all the images, icons, and logos used across the website.

## Getting Started

Follow these steps to set up the project locally on your machine:

1. **Install Dependencies:**
   Make sure you have [Node.js](https://nodejs.org/) installed. Then, open your terminal in the project folder and run:
   ```bash
   npm install
   ```
   This will install the required `sass` compiler.

2. **Run the SCSS Watcher:**
   To ensure that your changes in `style.scss` are automatically compiled into `style.css`, run the following command in your terminal:
   ```bash
   npx sass --watch styles/style.scss styles/style.css
   ```
   Keep this terminal tab open while you're working on the styling!

3. **View the Website:**
   You can view the website by simply opening `index.html` in your web browser. For the best development experience, it is recommended to use an extension like **Live Server** (if you use VS Code) to automatically refresh the page whenever you save a file.

## How to Make Updates

- **Updating Text or Images**: Open the relevant `.html` file (`index.html`, `about.html`, or `shop.html`) and make your changes. If you are adding a new image, place it in the `assets/` folder first.
- **Updating Colors, Fonts, or Styles**: Open `styles/style.scss`. You'll find variables for colors and fonts right at the top of the file, making it very easy to change the overall theme.
- **Updating Logic (Forms, Menus)**: Open `scripts/script.js` to modify how the website behaves interactively.

## Important Tips

- Always edit `style.scss`, **never** `style.css`.
- Ensure paths to images or other pages are relative (e.g., `href="about.html"` or `src="assets/images/logo.png"`) so that the site works smoothly both locally and when published online.
